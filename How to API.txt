1) You need to unzip the file and then open the terminal or whatever code editor you are using in this folder


2) install the required modules from the file 'requirements.txt' using pip. Or you can simply type the following command in terminal that will automatically install all the modules from requirements.txt file.

pip install -r requirements.txt


3) Run the app by typing the following command in terminal in the same directory

uvicorn main:app --reload


4) You can view the API on local host at port 8000, by typing the following URL in your browser

http://127.0.0.1:8000/


5) To see the docs, you can use the interface at the following link

http://127.0.0.1:8000/docs
